## 1. 개요

WOW는 팀 업무 현황을 한눈에 파악할 수 있는 주간 업무 관리 도구입니다.

- 담당자별 주간 일감을 격주 단위로 관리합니다.
- 팀원의 현재 상태(업무 중 / 휴가 / 종료)를 실시간으로 확인할 수 있습니다.
- 외부 관계자는 별도의 요약 화면에서 담당자 현황만 확인하고 Teams로 바로 연락할 수 있습니다.

---

## 2. 역할 구분

| 역할 | 설명 |
|------|------|
| **관리자** | 전체 기능 사용 가능. 담당자 추가/삭제/역할 변경, 타인 일감 이동 가능 |
| **팀원** | 본인 일감 및 상태 관리 가능. 타인 일감 조회 가능 |
| **외부인** | 팀 업무 요약(외부인 뷰)만 확인 가능. 주간 일감 상세 열람 불가 |

> 최초 가입 시 팀에 관리자가 이미 있으면 **외부인**으로 등록됩니다.
> 관리자가 담당자 관리 화면에서 역할을 변경해줘야 팀원/관리자로 승격됩니다.

## 3. 로그인

Microsoft 계정(회사 Azure AD)으로 로그인합니다.

- 로그인 버튼 클릭 → Microsoft 인증 팝업 → 완료
- 최초 로그인 시 자동으로 멤버 목록에 등록됩니다.

## 4. 상황판

화면 상단에 위치한 팀 현황 요약 패널입니다.

### 접기 / 펼치기
- **기본값**: 접힌 상태
- 헤더 영역을 클릭하면 펼쳐집니다.
- 접힌 상태에서도 **업무 중 N명 / 휴가 N명 / 종료 N명** 요약이 표시됩니다.

### 내 상태 변경
본인 행의 우측에서 상태를 변경할 수 있습니다.

| 상태 | 설명 |
|------|------|
| **업무 중** | 기본 상태. 초록 점으로 표시 |
| **휴가** | 클릭 시 휴가 종료일을 선택할 수 있습니다. 종료일이 지나면 자동으로 업무 중으로 전환됩니다 |
| **종료 📤** | 클릭 시 Teams 업무 종료 보고 화면이 열립니다. 다음 평일 아침에 자동으로 업무 중으로 전환됩니다 |

### 업무 설명
- 본인 행의 이름 아래 영역을 클릭하면 현재 업무를 간단히 입력할 수 있습니다.
- 입력 후 **Enter** 또는 클릭 영역 밖을 누르면 저장됩니다. (최대 60자)
- 외부인도 이 설명을 외부인 뷰에서 확인할 수 있습니다.

### 미보고 알림
평일 오후 6시 이후에도 업무 중 상태인 팀원에게 **⚠ 미보고** 표시가 나타납니다.

## 5. 주간 업무 보드

담당자별 주간 일감을 요일 단위로 관리합니다.

### 주차 이동
상단 WeekNav에서 이전 주 / 다음 주로 이동하거나 **오늘** 버튼으로 현재 주로 돌아올 수 있습니다.

### 보드 구성
- 각 담당자 섹션은 **월~금 5개의 컬럼**으로 구성됩니다.
- 각 컬럼에 해당 요일의 일감을 추가할 수 있습니다.
- 그룹이 설정된 경우 그룹 헤더로 구분되어 표시됩니다.

---

## 6. 일감 관리

### 일감 추가
- 각 요일 컬럼 하단의 **＋** 버튼 클릭
- 제목, 상태, 설명 입력 후 저장
- **여러 날 동시 추가**: 날짜 선택 모드에서 여러 요일 선택 가능
- **반복 추가**: 주 단위로 반복 등록 가능 (최대 수 주)

### 일감 상태
일감을 클릭하면 상태가 순환합니다.

| 상태 | 색상 |
|------|------|
| 할 일 | 회색 |
| 진행 중 | 파란색 |
| 완료 | 초록색 |
| 보류 | 주황색 |

### 일감 이동 (드래그 앤 드롭)
- 일감을 드래그하여 같은 담당자의 다른 요일로 이동할 수 있습니다.
- **관리자**는 다른 담당자의 컬럼으로도 이동할 수 있습니다.

### 일감 복사 (관리자 전용)
- 일감 우측 📋 버튼 클릭
- 복사할 대상 담당자, 주차, 요일 선택 후 복사

## 7. 이월 업무

완료되지 않은 업무를 다음 주로 넘길 때 사용합니다.

- 각 담당자 섹션 하단 **＋ 이월** 버튼으로 추가
- 이월 업무는 주황색 배경으로 구분 표시됩니다.
- 드래그 앤 드롭으로 순서 변경 가능

---

## 8. 담당자 관리

> 관리자만 접근 가능합니다.

헤더의 **👤 담당자 관리** 버튼을 클릭합니다.

### 담당자 추가
담당자 섹션 헤더의 ✚ 버튼 또는 담당자 관리 모달에서 추가합니다.

| 항목 | 설명 |
|------|------|
| 이름 | 필수 |
| 직급 | 필수 |
| 이모지 | 프로필 아이콘 |
| 부서/그룹 | 그룹 구분용 (선택) |
| 태그 | 외부인 뷰 필터용 (선택, 복수 입력 가능) |

### 순서 변경
담당자 관리 모달에서 **⠿ 핸들을 드래그**하여 순서를 바꿀 수 있습니다.

### 역할 변경
담당자 관리 모달에서 각 담당자의 역할(관리자 / 팀원 / 외부인)을 드롭다운으로 변경합니다.
단, 관리자가 1명뿐인 경우 마지막 관리자는 변경할 수 없습니다.

### 외부인 섹션
외부인으로 등록된 멤버는 별도 섹션(보라색 구분선)으로 분리되어 표시됩니다.

---

## 9. 그룹 & 태그

### 그룹
- 담당자 추가/수정 시 **부서/그룹** 항목 입력
- 같은 그룹의 담당자는 보드에서 그룹 헤더로 묶여 표시됩니다.

### 태그
- 담당자 추가/수정 시 **태그** 항목에 입력 후 Enter
- 기존에 등록된 태그는 입력 시 자동완성으로 표시됩니다. (↑↓ 방향키 + Enter로 선택)
- 태그는 외부인 뷰 상단의 **태그 필터**에 자동으로 표시됩니다.
- 여러 태그 동시 선택 시 AND 조건으로 필터링됩니다.

## 10. Teams 연동

### 업무 종료 보고
1. 상황판에서 **종료 📤** 버튼 클릭
2. 오늘 완료한 업무 목록이 자동으로 채워진 보고 양식이 열립니다.
3. 내용을 자유롭게 수정한 후 **Teams 전송 후 종료** 클릭
4. 전송 완료 후 상태가 **종료**로 변경됩니다.

> **종료만 처리**: Teams 전송 없이 상태만 종료로 변경합니다.
> **취소**: 보고 화면을 닫고 상태는 유지합니다(업무 중 유지).

## 11. 외부인 뷰

외부인으로 로그인하면 주간 업무 보드 대신 요약 카드 화면이 표시됩니다.

### 화면 구성
- 상단 배너: 현재 업무 중인 인원 수
- 멤버 카드: 이모지, 이름, 직급, 상태, 업무 설명, 태그
- 상태 순서: 업무 중 → 휴가 → 종료 순으로 정렬
- 종료 상태의 멤버는 60% 투명도로 표시

### 태그 필터
- 뷰 상단의 태그 버튼을 클릭하여 특정 태그의 담당자만 표시할 수 있습니다.
- **✕ 초기화** 버튼으로 필터를 해제합니다.

### Teams로 연락
- 이메일이 등록된 업무 중 담당자 카드 하단에 메시지 입력창이 표시됩니다.
- 메시지를 입력하고 💬 버튼(또는 Enter)을 누르면 Teams 앱이 열리면서 메시지가 입력된 상태로 시작됩니다.

## 12. 관리자 전용 기능

### 외부인 뷰 미리보기
헤더의 **👁 외부인 뷰** 버튼으로 외부인이 보는 화면을 미리 확인할 수 있습니다.
**📋 일반 보기** 버튼을 누르면 원래 화면으로 돌아옵니다.

### 타 담당자 일감 이동
관리자는 드래그 앤 드롭으로 다른 담당자의 컬럼에 일감을 이동할 수 있습니다.

### 타 담당자 일감 복사
일감 우측의 📋 버튼으로 원하는 담당자/주차/요일에 일감을 복사할 수 있습니다.
